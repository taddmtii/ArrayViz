Language Docs:

Features:
- func definitions and calls
- loops (while / for)
- control flow
- basic arithmetic
- variable access
- array operations
- built in function calls (such as print(), len(), abs(), sum(), sorted() etc...)

Strech goals:
- multiple assignments (a, b = 5, 6)
- visualize stack frames/heap
- Type annotations
- List comprehensions

Excluded:
- classes
- exeception
- generators
- decorators
- dictionaries
- bitwise operators
- global variables / keyword
